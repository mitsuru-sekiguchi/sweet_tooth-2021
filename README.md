# SweetTooth

## 概要
スイーツ専門の投稿サイトでユーザー登録をすることにより投稿が可能になる。
その他、投稿へのコメントやYummy!（いいね）で評価することが可能。

## 機能
#### ・ユーザー管理機能
#### ・投稿機能
#### ・投稿編集機能
#### ・投稿削除機能
#### ・コメント機能
#### ・yummy！（いいね）機能

## 目指した課題解決
#### ・スイーツに特化したSNSが欲しい
#### ・画像はあるが再度、購入・利用したいと思った時に商品名、金額、店舗名など情報が思い出せない
#### ・最新スイーツの情報収集や投稿で気になったスイーツがある店舗名、その金額が知りたい

## 要件定義
|機能                 |目的                               |詳細|ユーザーストーリー|
|--------------------|-----------------------------------|---|---------------------------------------------------------------------------|
|ユーザー管理機能       |ユーザー登録をするため                  |ユーザー情報を入力し登録する。-登録したユーザーが登録したメールアドレス、パスワードのみでログインができる。|・ユーザー情報を登録していないと投稿、編集、削除、コメントができな<br>・ユーザー情報を登録していると投稿、編集、削除、コメントができる|
|                    |                                   |
|投稿機能             |投稿をするため                        |投稿したい画像を選択し、商品名、金額、購入又は利用店舗、投稿に対してのコメント（任意）を入力し投稿ボタンを押すと投稿することができる。|・ユーザー登録をしていないと投稿ボタンが表示されない-・ユーザー登録をすると投稿ボタンが表示され、クリックすると投稿画面に遷移できる-・ユーザー登録をしていると投稿することができる|
|                    |                                   |
|投稿詳細機能          |投稿の詳細を確認できるようにするため        |投稿画面をクリックすると商品の画像、商品名、金額、購入又は利用店舗、投稿に対してのコメント、他ユーザーからのコメントの詳細を確認することができる。|・ユーザー登録の有無に関わらず、投稿画像をクリックすると投稿詳細ページに遷移する-・ユーザー登録していると投稿詳細画面でコメント入力欄が表示される-・ユーザー登録しているユーザーかつその投稿者であれば編集、削除ボタンが表示される-・ユーザー登録していないユーザーは投稿詳細画面に編集、削除ボタン、コメント欄が表示されない|
|                    |                                   |
|投稿編集機能          |投稿の編集をできるようにするため           |投稿詳細画面の編集をクリックすると商品の画像、商品名、金額、購入又は利用店舗、投稿に対してのコメントが編集することができる。|・ユーザー登録をしているユーザーかつ投稿者であれば投稿の編集ができる-・ユーザー登録をしているがその投稿者でなければ投稿の編集はできない-・ユーザー登録をしていないと投稿の編集はできない|
|                    |                                   |
|投稿削除機能          |投稿の削除をできるようにするため           |投稿詳細画面の編集をクリックすると投稿を削除することができる。|・ユーザー登録をしているユーザーかつ投稿者であれば投稿の削除ができる-・ユーザー登録をしているがその投稿者でなければ投稿の削除はできない-・ユーザー登録をしていないと投稿の削除はできない|
|                    |                                   |
|投稿一覧機能          |ユーザーの投稿一覧を確認できるようにするため |投稿一覧をクリックするとそのユーザーの投稿一覧を確認することができる。|・ユーザー登録の有無に関わらず投稿一覧を確認することができる|
|                    |                                   |
|投稿コメント機能        |投稿にコメントできるようにするため           |投稿詳細画面のコメント欄に入力をしてコメントボタンを押すとコメントができる。|・ユーザー登録をしていないとコメントができない-・ユーザー登録をするとコメントができる-・ユーザー登録をしていると投稿者でなくても投稿することができる|
|                    |                                   |
|投稿検索機能          |投稿を検索できるようにするため            |投稿検索欄にキーワードを入力すると検索することができる|・ユーザー登録の有無に関わらず投稿の検索することができる|
|                    |                                   |
|投稿Yummy!(いいね機能) |投稿に評価できるようにするため            |投稿詳細のいいねボタンを押すと数が表示され投稿の評価ができる|・ユーザー登録をしていなといいねが押せない-・ユーザー登録をしているといいねが押せる-・ユーザー登録をしているといいねの解除ができる|



 

## users テーブル

| Column        | Type      | Options     |
| --------------| ----------| ------------|
| nickname      | string    | null: false |
| email         | string    | null: false |

### Association

- has_many :sweets
- has_many :comments

## sweets テーブル

| Column          | Type       | Options          |
| ----------------| -----------| -----------------|
| name            | string     | null: false      |
| price           | integer    | null: false      |
| shop_info       | string     | null: false      |
| text            | text       |                  |
| user            | references | foreign_key:true |


### Association

- belongs_to :user
- has_many   :comments

| Column          | Type       | Options          |
| ----------------| -----------| -----------------|

| user            | references | foreign_key:true |
| sweet            | references | foreign_key:true |

- belongs_to :user
- belongs_to :sweet
